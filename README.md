# Запуск проекта


## Запуск сервера

Из корневой папки прописываем:

```
cd tech-int3-server
npm install
npm start
```

## Запуск сайта

Из корневой папки прописываем:

```
cd frontend
npm install
npm run dev
```

## Использованные библиотеки: 

### axios
Библиотека для удобных http-запросов, была поставлена для удобного использования АПИ

### recharts 
Библиотека для графиков на странице статистикию

### tanstack query
Библиотека была добавлена для кэширования и оптимизации работы c сервером
