# Запуск проекта


## Запуск сервера

Из корневой папки прописываем:

```
cd tech-int3-server
npm install
npm start
```

## Запуск сайта

Из корневой папки прописываем:

```
cd frontend
npm install
npm run dev
```

## Использованные библиотеки: 

### axios
Библиотека для обёртки http-запросов

### recharts 
Библиотека для графиков на странице статистики

### tanstack query
Библиотека была добавлена для оптимизации работы c сервером
